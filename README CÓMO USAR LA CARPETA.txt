En cada apartado se encuentra la información necesaria para cada apartado.

Lo que genera el apartadoX.c lo usa todo o casi todo el notebook de jupiter.

En el notebook de jupiter está TODO, pero no todo se puede usar. 
Si por ejemplo la presión se calcula solo en el apartado 3, entonces solo se puede ver la linealidad T-P.

Pero si en el apartado se pide 'mismo analisis que en apartado X', pues entonces PUEDES ver de nuevo el apartado X.


Por tanto, si se pide histograma, entonces se hace el histograma, si se pide un gráfico, se hace ese gráfico.
El orden de estos programas es el orden de aparición en el guión del ejercicio voluntario.
